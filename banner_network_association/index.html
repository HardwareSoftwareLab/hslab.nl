Banner Site Rules

of course it rules

#1 Embed the glue

You have to include this javascript:

<script type="text/javascript" src="../_misc/bna.js"></script>

#2 Include atleast one banner container

You have to include atleast one div for a banner:

<div class="banners_512x128"></div>

#3 Make a banner image

You have to make atleast one banner, so people can find your site.

#4 KISS-principe

Keep it simple, stupid
